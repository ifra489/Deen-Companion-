#  Deen Companion

Deen Companion is a modern Android application designed to help Muslims with their daily religious activities. It provides an elegant and user-friendly interface built with modern Android development technologies.

## ✨ Features

- 🔐 User Authentication (Firebase)
- 📖 Quran with Surah List & Details
- 🤲 Daily Duas
- 🕋 Prayer Times
- 🧭 Qibla Direction
- 📿 Digital Tasbeeh Counter
- 🕌 Qaza Namaz Tracker
- 💰 Zakat Calculator
- ⚙️ Settings
- 🌙 Material Design 3 UI


## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- MVVM Architecture
- Clean Architecture
- Dagger Hilt
- Room Database
- Firebase Authentication
- Firebase Firestore
- Coroutines & Flow
- Navigation Compose
- Material Design 3

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/deen-companion.git
```

2. Open the project in Android Studio.

3. Add your `google-services.json` file inside the `app/` folder.

4. Sync Gradle.

5. Run the application.



## 📌 Architecture

This project follows **MVVM + Clean Architecture**.

- Data Layer
- Domain Layer
- Presentation Layer
